# Data Course 💻

Цей репозиторій містить мої лабораторні роботи з курсу роботи з даними (Data Course). Усі завдання виконані мовою програмування **C++**.

## 📂 Структура репозиторію

Наразі в репозиторії представлені наступні роботи:

* **`lab1/`** — матеріали та вихідний код для Лабораторної роботи №1.
* **`lab2/`** — матеріали та вихідний код для Лабораторної роботи №2.
* **`lab3/`** — матеріали та вихідний код для Лабораторної роботи №3.
* **`Lab4_var1/`** — Лабораторна робота №4 (виконана за варіантом 1).
